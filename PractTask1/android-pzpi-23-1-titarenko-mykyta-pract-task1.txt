ХІД РОБОТИ

1.	Створення нового Android проєкту
1.1.	Відкриваємо Android Studio та натискаємо на кнопку "Start a new Android Studio project”.
1.2.	Обираємо шаблон для створення мінімального проєкту: - Обераємо Empty Activity як шаблон для застосунку і натискаємо Next.
1.3.	Вказуємо основну інформацію про застосунок: - Назва проєкту: task1. - Ім'я пакету: mykyta.titarenko.nure.- Місце збереження проєкту. - Мова програмування: Java. - Мінімальна версія SDK.
1.4.	Натискаємо Finish для завершення створення проєкту.
2.	Ознайомлення зі структурою проєкту
2.1.	Відкриваємо Android Studio та натискаємо на кнопку "Start a new Android Studio project”.
2.2.	Обираємо шаблон для створення мінімального проєкту: - Обераємо Empty Activity як шаблон для застосунку і натискаємо Next.
2.3.	Вказуємо основну інформацію про застосунок: - Назва проєкту: task1. - Ім'я пакету: mykyta.titarenko.nure.- Місце збереження проєкту. - Мова програмування: Java. - Мінімальна версія SDK.
2.4.	Натискаємо Finish для завершення створення проєкту.
3.	Налаштування емулятора або підключення фізичного пристрою
3.1.	 Використовуємо Android Virtual Device (AVD) Manager, щоб налаштувати віртуальний пристрій.
3.2.	Підключаємо фізичний пристрій (телефон або планшет) через USB і вмикаємо режим налагодження (Developer Mode).
4.	Запуск мінімального застосунку
4.1.	Натисніть кнопку Run для компіляції та запуску застосунку.
4.2.	Обираємо емулятор для запуску.
4.3.	Бачимо, що на екрані відображено простий інтерфейс із елементом TextView.
5.	Внесення змін до інтерфейсу
5.1.	Відкриваємо файл activity_main.xml у редакторі та додаємо кнопку (Button).
5.2.	Встановлюємо текст кнопки за допомогою атрибуту android:text="Натисніть мене". - Встановлюємо розміри кнопки, використовуючи android:layout_width та android:layout_height.
5.3.	Запускаємо застосунок ще раз і бачимо кнопку з написом “Натисніть мене”.
6.	Взаємодія з інтерфейсом
6.1.	У файлі MainActivity.java/MainActivity.kt додаємо обробку натискання на кнопку: - Знаходимо кнопку за її ідентифікатором за допомогою методу findViewById(). - Додаємо обробник натискання за допомогою методу setOnClickListener().
6.2.	Змінюємо текст елемента TextView в методі onClick за допомогою setText.
6.3.	Запускаємо застосунок ще раз. Бачимо, що при натисненні на кнопку, текст елемента TextView змінюється.
